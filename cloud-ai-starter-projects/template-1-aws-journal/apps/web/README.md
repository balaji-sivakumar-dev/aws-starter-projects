# Web App

React SPA for Template 1 journal starter.

## Features
- Cognito Hosted UI login/logout (Authorization Code + PKCE)
- Profile (`/me`) display
- Entry list + detail
- Create/edit/delete entry
- Trigger AI processing and view `aiStatus`, `summary`, and `tags`

## Local Run
```bash
cd apps/web
cp .env.example .env
npm install
npm run dev
```
